# Forum API
Exercício do curso Spring Boot Parte 2: Segurança da API, Cache e Monitoramento da Alura


### Stack Utilizada
* Java 8
* Spring Boot
* Spring Data Jpa
* Spring Cache
* Spring Security
* Spring Actuator
* Spring Boot Admin
* Spring Devtools
* SpringFox Swagger
* H2 database
* JWT
